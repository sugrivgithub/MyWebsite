What is Cypress?

Cypress is an open-source testing framework that helps you write fast, reliable, and easy-to-understand tests for your web applications.

With Cypress, you can automate end-to-end tests, integration tests, and unit tests for your web apps.

Getting Started with Cypress

This section will help you set up Cypress and write your first test!

Step 1: Install Cypress

Run the following command in your terminal:

npm install cypress --save-dev

Step 2: Write Your First Test

Create a test file in cypress/integration and write your first test:

describe('My First Test', () => {
    it('Visits the Cypress website', () => {
      cy.visit('https://www.cypress.io');
      cy.contains('Docs').click();
      cy.url().should('include', '/docs');
    });
  });

Working with Selectors

Selectors help you target elements on your web page.

cy.get('#button-id').click();
cy.get('.button-class').should('be.visible');

Handling Elements

Clicking a Button

cy.get('button').click();

Typing in an Input Field

cy.get('input').type('Hello Cypress!');

Dragging and Dropping

cy.get('#drag-item').drag('#drop-area');

Assertions in Cypress

cy.get('h1').should('contain', 'Welcome');
cy.url().should('include', '/docs');

Working with APIs

Using cy.request()

cy.request('https://api.example.com/data').then((response) => {
      expect(response.status).to.eq(200);
    });

Handling Multiple Windows

cy.window().then((window) => {
      // Use the window object to manipulate multiple windows
    });

Creating Custom Commands

Create reusable custom commands to simplify your tests.

Cypress.Commands.add('login', (username, password) => {
    cy.get('#username').type(username);
    cy.get('#password').type(password);
    cy.get('#login-button').click();
  });

Page Object Model (POM)

Organize tests with Page Object Model to separate test logic from UI interactions.

class LoginPage {
    visit() {
      cy.visit('/login');
    }
  
    fillCredentials(username, password) {
      cy.get('#username').type(username);
      cy.get('#password').type(password);
    }
  
    submit() {
      cy.get('#login-button').click();
    }
  }
  
  const loginPage = new LoginPage();
  loginPage.visit();
  loginPage.fillCredentials('user', 'password');
  loginPage.submit();
